Tweener

public final class Tweener

A singleton, factory class to create, queue, and manage Tween animations.

Factory Creation
  • animate(_:to:duration:completion:)

    Creates a Tween that animates an array of properties on a Tweenable type to desired values from its current values over a set duration.

    The created tween will automatically be queued to start if Defaults.autoStartTweens is true.

    Declaration

    Swift

    @discardableResult
public func animate<Property: TweenableProperty>(
    _ target: Property.Target, to properties: [Property],
    duration: TimeInterval, completion: Tween.Callback? = nil) -> Tween

    Parameters

    target

    The object on which properties are animated.
    properties

    The array of properties to be animated.
    duration

    The amount of seconds the animation takes to complete.
    completion

    The callback invoked when the animation is finished (optional).

    Return Value

    The Tween control for the animation.

Tweenable

public protocol Tweenable

A type that can animate tweenable properties on itself.

  • tween(to:duration:completion:)

    Animates a tweenable property from its current value to a desired value.

    Declaration

    Swift

    @discardableResult
func tween<Property>(to property: Property, duration: TimeInterval, completion: Tween.Callback?) -> Tween where Self == Property.Target, Property : TweenableProperty

    Parameters

    property

    The property to animate.
    duration

    The amount of seconds the animation takes to complete.
    completion

    The callback invoked when the animation is finished (optional).

    Return Value

    The Tween control for the animation.

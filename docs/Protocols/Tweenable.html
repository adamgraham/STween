Tweenable

public protocol Tweenable

A protocol to provide tweening animation functionality.

Methods

    A method to create an animation on self animating to a tweenable property value.

    Declaration

    Swift

    func tween<TweenProperty: TweenableProperty>(to property: TweenProperty, duration: TimeInterval, completion: Callback?) -> Tween

    Parameters

    property

    The property to be animated.
    duration

    The amount of time, in seconds, the animation will run.
    completion

    A callback invoked when the animation is finished.

protocol TweenTimerDelegate: class

A protocol to inform a TweenTimer‘s delegate of events.

  • tweenTimer(_:didUpdateWithElapsedTime:)

    A delegate method to inform the listener that a timer invoked a tick event and has updated the elapsed time.

    • Parameters:
      • timer: The timer the tick event was invoked on.
      • elapsed: The amount of time, in seconds, the timer has been running.
